How to Use

  • Use Time Zones to view UTC and your local time.
  • Click + Add Timezone to add extra time zones.
  • Select a city from the dropdown; it will display the 3-letter airport code.
  • To remove a timezone: tap the timezone line to select it, then press Delete (desktop) or tap Remove (mobile).

Features

  • View MEL in UTC or Local time via the MEL time base toggle.
  • View or hide the Calculator from the toolbar.
  • Font size can be adjusted with sliders; the sliders can be toggled on or off.
  • Theme can be switched between Dark and Light modes.

Panels and Toggles

  • Use the toolbar toggles to show/hide the Date Offset panel and Calculator.
  • The Date Offset panel is off by default and can be enabled from the toolbar or mobile button.
  • Panels can be dragged and resized.

Font Sliders

  • Font sliders are hidden by default. Use the toggle to show them.
